# 📊 Excel Sales Dashboard Project

## 📌 Project Overview
This project is an interactive Sales Dashboard created entirely in Microsoft Excel. The dashboard provides insights into sales performance by analyzing data for different cities and sales executives. It helps in identifying top performers, bottom performers, and target achievement percentages through dynamic visualizations.

## 🎯 Objective
The objective of this project is to transform raw sales data into meaningful insights using Excel features and automation techniques.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Formulas and Functions
- VBA Macros
- Data Validation
- Dashboard Design

## 📈 Dashboard Features
- Interactive dashboard with city-wise filters.
- Sales Executive-wise analysis.
- Total Sales comparison.
- Target Achievement Percentage analysis.
- Top Performers identification.
- Bottom 5 Performers analysis.
- Dynamic charts and graphs.
- User-friendly navigation using slicers.
- Automated report updates using Macros.

## 📊 Visualizations Included
- Bar Charts
- Pie Charts
- Pivot Charts
- Percentage Analysis
- Sales Performance Comparison

## 🔄 Automation
This dashboard uses Excel VBA Macros to automate report generation and improve user interaction. The dashboard updates dynamically based on selected filters and slicers.

## 📂 Dataset
The dataset contains information about:
- Sales Executives
- City Names
- Total Sales
- Target Achievement Percentage
- Performance Metrics
